# **Basics of HTTP/HTTPS**

## **1. Overview**
**HTTP (Hypertext Transfer Protocol)** enables web communication, while **HTTPS (HTTP Secure)** adds SSL/TLS encryption for security.

### **Key Differences**
| Feature | HTTP | HTTPS |
|---------|------|-------|
| Security | No encryption | Encrypted with SSL/TLS |
| Data Protection | Vulnerable | Secured |
| Usage | General websites | Banking, e-commerce |

## **2. HTTP Request & Response Structure**
### **Request Structure**
- **Method** (GET, POST, etc.)
- **Headers** (Metadata)
- **Body** (Optional, used in POST/PUT)

### **Response Structure**
- **Status Code** (200 OK, 404 Not Found)
- **Headers** (Content-Type, Cookies)
- **Body** (HTML, JSON, etc.)

## **3. Common HTTP Methods**
| Method | Description |
|--------|-------------|
| GET | Retrieve data |
| POST | Submit data |
| PUT | Update data |
| DELETE | Remove data |

## **4. Common HTTP Status Codes**
| Code | Meaning |
|------|---------|
| 200 | OK |
| 301 | Moved Permanently |
| 404 | Not Found |
| 500 | Server Error |

## **5. Tools for Analysis**
- **Browser DevTools** (Inspect Network tab)
- **Wireshark** (Analyze HTTP/HTTPS traffic)
